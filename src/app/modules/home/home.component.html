The Sonnet Database and Toolkit

A Corpus of English Sonnets for Text Analysis

Just what is a sonnet? Definitions have focused on its form: fourteen lines of rhymed pentameter. But early sonnet forms are more variable than that. So other definitions focus on its rhetoric, genre, or style: a first-person reflection with a volta or turn from problem to resolution.

This project addresses the question by building a corpus of every English sonnet from the sixteenth century onward, in order to quantify their features through time. We offer a RESTful API interface for text analysis.

Text Encoding Initiative Encoding all the texts.

The Text Encoding Initiative (TEI) is a consortium which collectively develops and maintains a standard for the representation of texts in digital form. Its chief deliverable is a set of Guidelines which specify encoding methods for machine-readable texts, chiefly in the humanities, social sciences and linguistics.

Augmented Criticism Lab The vision and main contributors.

The Augmented Criticism Lab is a digital initiative that investigates the relationship between computational literary studies and traditional philology. Python Database Connector Powering programmatic analysis.

We provide and maintain a Python class for easily connecting to our database via the RESTful API. This class provides basic query functions and JSON object serialization. This allows a user to skip most of the data parsing and jump straight into NLTK or NLP analysis of the data.
